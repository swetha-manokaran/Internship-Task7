# Web Application Security Assessment using OWASP ZAP

## Objective
To identify security vulnerabilities in a web application using OWASP ZAP.

## Tool Used
- OWASP ZAP
- Kali Linux

## Target
- http://testphp.vulnweb.com (Authorized test website)

## Methodology
- Automated scan using OWASP ZAP
- Passive and active scanning performed
- Vulnerabilities identified and documented

## Outcome
The scan revealed multiple vulnerabilities of varying risk levels including
High, Medium, Low, and Informational issues. The generated HTML report contains
detailed findings and remediation suggestions.
